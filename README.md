# О чём проект
Проект содержит решение задачи, предложенной для прохождения промежуточной аттестации перед выбором специализации.
Описание задачи включено в текст кода: *Написать программу, которая из имеющегося массива строк*
*формирует новый массив из строк, длина которых меньше, либо равна 3 символам.*
*Первоначальный массив можно ввести с клавиатуры, либо задать на старте выполнения алгоритма.*
*При решении не рекомендуется пользоваться коллекциями, лучше обойтись исключительно массивами.*
# Краткое описание решения
Решение приведено в виде функции. Для проверки использована еще одна функция, 
заимствованная из решения задачи на семинаре (вывод массива в консоль).
Обрабатываемый массив не вводится пользователем с консоли, а задаётся перед вызовом функции.
* Первым этапом обработки массива является подсчёт элементов, имеющих размер меньше или равный трём символам. 
Это необходимо для определения размера массива, в который эти элементы впоследствии будут помещены.
* Второй этап - с использованием найденного значения создаётся собственно новый массив, и уже в него
посредством перебора в цикле помещаются элементы, размер которых меньше или равен трём символам.
* Функция возвращает этот новый массив.
